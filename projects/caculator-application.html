calculator application

A simple calculator built with JavaScript, HTML, and CSS. It performs basic arithmetic operations—addition, subtraction, multiplication, and division—and features a responsive design for use on various devices. This project showcases front-end development skills and progress made during self-study of JavaScript.

Key Features:

  • Responsive Design: Ensures optimal functionality on various devices, including desktops, tablets, and smartphones.
  • Basic Arithmetic Operations: Supports addition, subtraction, multiplication, and division for straightforward calculations.
  • User-Friendly Interface: Features a clean and intuitive layout for easy navigation and usage.
  • Interactive Elements: Implements buttons and displays that enhance user experience and engagement.
  • Extensive Front-End Coding: Involves significant front-end development efforts to achieve the calculator format and functionality without using any CSS framework.

Technologies Used:

  • HTML: For structuring the calculator layout and content.
  • CSS: For styling and creating a responsive design without using any CSS framework.
  • JavaScript: For implementing interactive features and performing arithmetic calculations.
