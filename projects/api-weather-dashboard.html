This project is a Weather Dashboard built using HTML, CSS, and JavaScript, utilizing the OpenWeather API to provide real-time weather data for any city. Users can input a city name, and the dashboard will display key weather details such as temperature, humidity, wind speed, and overall weather conditions (e.g., sunny, cloudy). The project showcases the integration of third-party APIs with a responsive design, making it accessible on both desktop and mobile devices, while offering a practical learning experience in API handling and front-end development.

Key Features:

  • Real-time weather data fetching using the OpenWeather API.
  • Responsive design, accessible on both desktop and mobile devices.
  • City-based weather search functionality with easy input interface.
  • Displays key weather information including temperature, humidity, wind speed, and weather conditions.
  • Error handling for invalid city names or API failures.

Technologies Used:

  • HTML: Structure and layout of the dashboard.
  • CSS: Styling and responsive design.
  • JavaScript: Logic for fetching and displaying weather data.
  • OpenWeather API: Source of real-time weather data.
  • Fetch API: To make asynchronous HTTP requests for weather information.
