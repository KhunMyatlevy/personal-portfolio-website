To Do List Application

A simple and interactive to-do list app built using HTML, CSS, and JavaScript. It allows users to add, delete, and manage tasks efficiently. The app features a clean, user-friendly interface with real-time task updates and dynamic functionality.

Key Features:

  • Add and delete tasks with a single click
  • Real-time task updates and tracking
  • Clean, user-friendly interface
  • Fully responsive design for mobile and desktop
  • Persistent data using browser local storage

Technologies Used:

  • HTML for structuring the layout of the application
  • CSS for styling and responsive design
  • JavaScript for adding dynamic functionality and task management
