Memory Card Game

A fun and interactive memory card game where players flip cards to find matching pairs. Built using HTML, CSS, and JavaScript, the game tests the player's memory and improves focus. The cards flip dynamically, and the game tracks the player's progress until all pairs are found.

Key Features:

  • Interactive Gameplay: Flip the cards to find matching pairs and test your memory skills.
  • Dynamic Card Flip Animation: Smooth card flipping effect for a visually appealing experience.
  • Progress Tracking: Keeps track of matched pairs until all are found.
  • Responsive Design: Works seamlessly on different screen sizes and devices.
  • Customizable Layout: Easily change the number of card pairs or the design of the cards.

Technologies Used:

  • HTML: Structured the layout of the game board and cards.
  • CSS: Styled the game elements and added animations for the card flipping effect.
  • JavaScript: Implemented the game logic, card flipping, matching functionality, and user interactions.
